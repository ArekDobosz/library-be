## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

Add .env file to the root of the repository then run in console:

```bash
$ npm install
$ docker compose build
$ docker compose up
$ npm run db:generate
$ npm run db:push
$ npm run db:seed
$ npm run start
```

## Test

```bash
$ npm run test
```
